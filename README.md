# 고고 오락실 (GogoPlayRoom)

Three.js 기반 3D 아케이드 게임룸입니다. 브라우저에서 실행되며, 3D로 구현된 오락실 공간에서 게임을 선택하고 플레이할 수 있습니다.

![메인 화면](docs/main.png)

## 게임 목록

| 게임 | 장르 |
|------|------|
| 고고드라이브 | 3D 레이싱 |
| 베스트 슈터 | 2D 슈팅 |
| 로봇 파이터 | 3D 액션 |

## 플레이 방법

1. **동전 추가** - 우측 상단 `+ 동전` 버튼 클릭 → 4자리 비밀번호 입력 → 수량 선택
2. **동전 투입** - 오락기 하단의 코인 슬롯 클릭 (동전이 있으면 마우스 커서가 회전하는 3D 동전으로 변경)
3. **게임 시작** - `PRESS START` 깜빡일 때 스크린 클릭
4. **게임 종료** - `ESC` 키 또는 `EXIT` 버튼

![동전 추가](docs/coin-modal.png)

## 기술 스택

- **Three.js** (r128) - 3D 렌더링, 절차적 모델링
- **Web Audio API** - 효과음
- **localStorage** - 동전 수량 저장
- HTML / CSS / Vanilla JS

## 3D 씬 구성

- 절차적으로 생성된 아케이드 캐비닛 (마키, CRT 스크린, 조이스틱, 버튼, 코인 슬롯)
- 벽면 포스터, 테이블, 스툴, 껌볼 머신, 휴지통, 커비 오브젝트
- 브라운 카펫 바닥, 벽돌 패턴 벽면
- 마우스 호버 시 오락기 확대 효과
- 동전 투입/낙하 애니메이션

## 실행

로컬 HTTP 서버로 실행합니다.

```bash
python3 -m http.server 8080
# http://localhost:8080 접속
```
